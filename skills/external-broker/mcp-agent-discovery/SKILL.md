---
name: mcp-agent-discovery
description: Discover external agents through the MCP catalog, then narrow candidates by category, capability, and task fit.
---

Use this skill when the broker needs to find a suitable external agent.

Guidelines:
- Start with discovery and candidate listing before delegation.
- Filter by category when the task has an obvious domain, such as compliance, security, or analytics.
- Filter by capability when the task asks for a specific kind of output.
- If multiple candidates remain, prefer the one with the clearest capability match and the narrowest specialty.
- If no strong match exists, explain that gap instead of pretending one agent is perfect.
